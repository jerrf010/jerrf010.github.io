What is a div element?

A div element is a container used to group other HTML elements together.

What is one way you can identify or "target" a div element?

One way to target a div element is by using its class or id attribute in CSS.

Why would you use a div element?

Div elements are useful for applying styles to a group of elements, creating layout structures, and managing sections of a webpage.

What is a border property?

The border property in CSS is used to specify the border of an element. It can define the border's width, style, and color.

What are some of the different border properties?

  • border-width: Specifies the width of the border.
  • border-dotted: Specifies the style of the border as dotted.
  • border-color: Specifies the color of the border.
  • border-radius: Specifies the radius of the border's corners.

What is the difference between margin and padding?

Margins create space outside of an element, while padding creates space inside an element, between the element's content and its border.
